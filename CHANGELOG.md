# 更新日志

## [1.0.2] - 2024-12-04

### 🔥 重要更新

- **修复数据丢失问题**：将数据存储从 localStorage 迁移到文件系统持久化
- 数据现在保存在 `~/Library/Application Support/com.quick-note.app/note.txt`
- 应用重启后数据不再丢失

### ✨ 新增功能

- 输入时自动保存（500ms 防抖）
- 关闭面板时立即保存
- 应用退出前自动保存
- 启动时自动加载上次的内容

### 🛠️ 技术改进

- 添加 `tauri-plugin-fs` 依赖
- 实现 `save_note` 和 `load_note` 后端命令
- 优化保存逻辑，避免频繁写入

### 📝 文档更新

- 添加数据持久化说明文档
- 更新 README 中的技术栈说明
- 添加数据存储位置说明
